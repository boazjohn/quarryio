Import data into Quarry

To get started, you'll need to import some data into your quarry. You can import from many different sources with a few simple wizards and tools. Data imported into Quarry is stored securely using Amazon S3 and can be easily exported at any time in the future.

Data Source
External Database Connect an external database (such as MySQL, PostgreSQL, MongoDB or ElasticSearch) to be able to directly query it's data and run analytics on it. Database Connection Wizard
